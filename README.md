# DenoGrad-S
Using **sparsed deep learning models** as backbones for [DenoGrad](https://github.com/ari-dasci/S-DenoGrad) framework as they generalize better and do not fall into overfitting noisy data.

We expect to obtain, at least, the same results we had with default backbones in DenoGrad, but with a much faster inference and substantial less VRAM use.
